# geo411-log-models
Description: Using R Studio to compare linear models and log models on case study revolving car weight and miles per gallon (MPG).

Methods: Used R studio to generate graphs and summaries of datasets.

Key Findings: For every 1000lb increase in weight, MPG decreased by about 5.3, or for every 1% increase in weight, there was a decrease of 0.27 in MPG.

Tools: R Studio
